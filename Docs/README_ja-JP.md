## 📖はじめに
機能: ゲームバージョン 4.0 に対応
チャットグループに参加する: [Discord](https://discord.gg/castoriceps)

## 📦クイックスタート
要件: .NET 9.0 [ダウンロードはこちら](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)

```bash
git clone https://github.com/DBKAHHK/HyacineDH-Core.git
cd HyacineDH-Core
dotnet run

```

## 📝使用方法
ソースコードからコンパイルしたくない場合は、[リリース](https://github.com/DBKAHHK/HyacineDH-Core/releases) からコンパイル済みファイルをダウンロードしてください。
